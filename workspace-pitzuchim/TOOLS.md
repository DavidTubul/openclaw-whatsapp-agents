# TOOLS.md — הערות מקומיות

הסקילז מגדירים _איך_ הכלים עובדים. הקובץ הזה הוא ל_פרטים שלך_ — מה שייחודי להתקנה הזו.

## מה הולך כאן

- מספרי טלפון / סניפים של החנות (כשיאומתו)
- כתובות, שעות מיוחדות (חגים)
- קיצורי דרך / נהלים ייחודיים שתלמד

## הכלים שלי (ראה `skills/customer-service/SKILL.md` לפירוט)

- `node tools/cases.mjs ...` — יומן מקרי שירות/תלונות (append/list/read/update/export-csv/stats). מקור האמת המקומי: `data/cases/cases.jsonl`. אם `sheet.enabled=true` ב-`bot.json` — דוחף גם ל-Google Sheet.
- `node tools/lib/policy.mjs` — לוגיקת זכאות לזיכוי (חישוב ימים-עד-תוקף + החלטה). נבדק ביוניט.
- אימות תמונה = יכולת הראייה שלי (כלי Read על קובץ ב-`~/.openclaw/media/inbound/`), לא כלי דטרמיניסטי.
