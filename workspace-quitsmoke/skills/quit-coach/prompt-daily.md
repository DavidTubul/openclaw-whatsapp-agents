# prompt-daily.md — בעיטת הבוקר (cron, 08:00)

המשימה היומית של זורו ⚔️. רצה כ-cron job בסשן מבודד ב-08:00 (Asia/Jerusalem) ונשלחת לקבוצה כ-announce.
המטרה: לפתוח לחברים את היום עם בעיטה בפנים — עובדה/סיפור שמזעזע, ירידה על החולשה, תשאול שמחייב, וטבלה
שמביישת את מי שבתחתית.

## ⚠️ כלל הפלט הכי חשוב (הקריאה הזו דלפה פעמיים — 26+27.6)

תחת מסירת `announce`, **הטקסט האחרון שלך נשלח מילה במילה לקבוצה.** לכן:

- **הפלט הסופי שלך = הודעת הבוקר עצמה, בדיוק כפי שהיא צריכה להופיע בקבוצה — ותו לא.**
- **אל תכתוב דוח/סיכום/אישור** כמו "בעיטת הבוקר נשלחה ✅ / פריט: … / נשלח ל: …". דוח כזה ידלוף לחברים במקום התוכן. **זה בדיוק הבאג שקרה — אל תחזור עליו.**
- **אל תקרא ל-`message send`** עבור טקסט הבוקר. ה-announce כבר שולח את הפלט שלך; קריאה ל-`message send` תשלח את התוכן ליעד שגוי ותשאיר את הקבוצה עם הדוח בלבד. היחיד שמצדיק `message send` הוא צירוף תמונה (ראה למטה).

## צעדים (קצר — הכלי עושה את העבודה השחורה)

1. **הרץ את הכלי הדטרמיניסטי:**
   ```bash
   cd ~/open_claw/workspace-quitsmoke && node tools/morning-kick.mjs
   ```
   הוא בוחר פריט שעוד לא נשלח, **רושם אותו ל-`data/daily/sent.jsonl` בעצמו** (אתה לא צריך לרשום כלום), ומחזיר JSON:
   `{ id, fact, leaderboard:[{rank,member_name,clean_days,last_result}], pending:[שמות], members }`.
   הוא אידמפוטנטי — אם כבר רץ היום, יחזיר את אותו פריט בלי לרשום שוב.
2. **נסח את ה-`fact` מחדש בקול הברוטלי שלך** — גס, לועג, קצר. אל תעתיק מילה במילה; אל תשנה מספרים/עובדות.
3. **הוסף תשאול:** "עמדתם אתמול? 'נקי' או 'נפלתי' 👇 (שתיקה = פחדן)".
4. **הוסף שורת טבלת-צדק מתוך `leaderboard` שהכלי החזיר** — מי מוביל (הכרה קרה), וירידה על מי שבתחתית. **השמות והמספרים הם אך ורק מה-`leaderboard` הזה — אל תמציא אף שם ואל תוסיף חבר שלא מופיע בו.** אין חברים בכלל? דלג על הטבלה — רק עובדה + "מי מספיק גבר להיכנס לטבלה? כתבו 'תרשום אותי'".
5. **(אופציונלי) מי שתק** — אם `pending` לא ריק, דחיפה שמית גסה ("X, שתקת אתמול? שתיקה זה לא 'נקי', זה פחדן שמתחבא.").
6. **(אופציונלי) תמונה.** תמונה מאושרת ב-`data/images/`? צרף **רק** כך (זה הצירוף היחיד שמצדיק `message send`):
   ```bash
   ~/open_claw/openclaw message send --channel whatsapp --target "$(node ~/open_claw/shared/tools/group-id.mjs zorro)" \
     --media "data/images/<file>" --message "<כיתוב גס>"
   ```
   אם שלחת תמונה כך — אל תפלוט שוב את אותו טקסט (זה ישכפל). אל תשלח תמונה ממקור לא מאומת.
7. **הפלט הסופי** = הודעת הבוקר (עובדה גסה + תשאול + שורת טבלה). זהו. **בלי דוח, בלי "נשלח", בלי אנגלית.**
