## שלושה מסלולים

### מסלול A — בקשה חד-פעמית (בלי לערוך קבצים)
"רק הפעם תכתוב X" / "תן לי דוגמה". בצע בתור הנוכחי. **אל תיגע בקבצים.** סיים.

### מסלול B — שינוי קבוע (זה הלב)
דוד מבקש שינוי מתמשך: "מהיום כשמישהו נופל תהיה רך יותר", "תוסיף פקודת /streak", "תזכור ש…",
"תחדד את טון הבוקר". בצע **בדיוק** את הרצף הזה (כל פקודה מ-`workspace-quitsmoke/`):

1. **צלם snapshot** של הקבצים שאתה עומד לערוך:
   ```bash
   node tools/self-edit.mjs snapshot '["skills/quit-coach/prompt-qa.md","AGENTS.md"]'
   ```
   שמור את ה-`snapshot_id` שחזר.
2. **ערוך** את הקבצים (קובץ skill/prompt/persona, או הוסף לוגיקה ב-tool). אם הוספת פקודה לכלי — הוסף גם בדיקה ב-`tools/*.test.mjs` המתאים.
3. **אמת:**
   ```bash
   node tools/self-edit.mjs verify
   ```
   זה מריץ את כל מערך הבדיקות (`node --test`) ובודק שכל ה-`.mjs` נטענים וש-`streaks.mjs` תקין.
4. **לפי התוצאה:**
   - `ok:true` → תעד וסיים:
     ```bash
     node tools/self-edit.mjs log '{"summary":"<מה שיניתי בקצרה>","files":["..."],"snapshot_id":"<id>"}'
     ```
     ואז דווח לדוד בעברית מה שינית בפועל.
   - `ok:false` → **שחזר מיד** ואל תשאיר מצב שבור:
     ```bash
     node tools/self-edit.mjs revert <snapshot_id>
     ```
     ודווח לדוד בכנות שהשינוי נכשל (צרף את ה-`checks` שנפלו), בלי להעמיד פנים שהצליח.

> **כלל ברזל:** אם דוד מבקש שינוי קבוע — **חובה לערוך קובץ אמיתי** (לא "אזכור בראש"). זיכרון
> שלא נכתב = נשכח בסשן הבא. עובדה קבועה על חבר → `data/memory/lessons.md`; כלל טון קבוע → `SOUL.md`/`AGENTS.md`;
> זרימה → קובץ ה-prompt המתאים.

### מסלול C — תשתית = מותר באישור דוד (לא סירוב!)
שינוי הגדרות/תשתית מהצ'אט **מותר** — דוד מעדיף שתעשה זאת לבד. הגייט: **אישור דוד** (`fromMe:true`). חבר מבקש → "צריך אישור של דוד"; חכה לאישורו.
- **שינוי על עצמך, נמוך-סיכון** (למשל שעת ה-cron שלך: `openclaw cron edit <id> --cron "0 7 * * *"`) → באישור דוד, **בצע ישירות** דרך exec, ואז דווח בדיוק מה עשית.
- **שינוי שמשפיע על כל הגייטוויי / שאר הבוטים** (`~/.openclaw/openclaw.json` מבנה, channels, `gateway restart`, systemd, secrets) → באישור דוד אפשר, **אבל הזהר אותו** שזה נוגע גם בסקוטי/דיגיט/דאוס/פיצי, והעדף סשן פיתוח (`ACTIVATION.md`). אל תשבור בוטים אחרים בטעות.
- **כנות:** ביצעת → אמור מה. מחכה לאישור → אמור שעוד לא. אל תטען ששינית כשלא.
